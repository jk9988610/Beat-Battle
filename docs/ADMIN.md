# 管理员说明

## 打开设置

点击顶栏 **⚙️** 或访问 `#settings`。

## 指定管理员

在 **设置 → 管理员设置**（需先验证身份）：

| 项 | 说明 |
|----|------|
| **管理员昵称** | 每行一个；用这些昵称加入赛季即为管理员 |
| **管理员口令** | 任意昵称登录后，在「管理员身份」输入口令可临时获得权限 |

配置保存在**本浏览器**，修改后请通知其他主持人。

## 首次使用

默认昵称：`管理员`、`主持人`；默认口令：`bb2026`。请尽快在设置中修改口令。

## 管理员权限

- 主页「活动管理」：推进赛季阶段、结束赛季、导入导出
- 设置页：修改管理员昵称与口令

## 安全说明

前端校验，适合活动场景。请勿将 `service_role` 写入网页。
